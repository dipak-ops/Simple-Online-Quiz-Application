The Simple Online Quiz Application is a web-based project created for beginners who want to learn frontend web development.<br> This project allows users to answer multiple-choice questions and get instant results. It is simple, interactive, and easy to understand.

This project is a quiz website where questions are shown one by one with four options.<br> The user selects an answer and clicks the Next button to continue.<br> After all questions are completed, the final score is displayed on the screen.

The project is developed using HTML, CSS, and JavaScript.<br> HTML is used to create the structure of the web page, CSS is used to design and style the quiz interface,<br> and JavaScript is used to handle the quiz logic such as loading questions, checking answers, ,br>calculating scores, and showing results without reloading the page.
